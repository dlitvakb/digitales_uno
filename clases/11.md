# Sumador/Restador

```
      |Y      |M
      |       |
  |---|-------|----|
  |   |       |    |   S = Y  Si M = 0
  |    \      /    |   S = ~Y Si M = 1
  |   \_|____|_/   |
  |  \\________//  |
  |   \________/   |
  |________|_______|
           |S

La XOR es una NOT controlada
```
