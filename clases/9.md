# Multiplexores (MUX)

## Diagrama Logico

```
Datos _______  Salida
  ___|E3     |
  ___|E2     |
  ___|E1    S|____
  ___|E0     |
     | C0 C1 |
     |__|_|__|
        | |
     Control de
     Seleccion
```

## Tabla de verdad (abreviada)

```
C1 | C0 || S
 0 |  0 || E0
 0 |  1 || E1
 1 |  0 || E2
 1 |  1 || E3
```

## Expresion logica

`S = (~C1~C0)E0 + (~C1C0)E1 + (C1~C0)E2 + (C1C0)E3`
